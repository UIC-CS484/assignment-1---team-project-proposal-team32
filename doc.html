myWeather Unlimited

We will use the Open Weather Map API.


This API will be used to retrieve the current weather and future forecast of the user's entered location. This should allow the user to plan for today and ahead, such as clothing to wear and outdoor plans.


From the user, we will collect login information and a list of his favorite location (that he will save to read the weather from).


USER -> id, fname, lname, email, password, current-location; FAVORITES -> id, favorited-location

This app shall be used by commuters, vacationers, people planning outdoor activities, and travellers.


This site aims to:
(1) Help user make choices according to the current and future forecasts
(2) Show the user the forecasts of his favorite'd locations upon login


We wanted to work with a simple API so we could learn the basics of web-dev easily. We also chose this API because it is something that can be readily used by anyone.
